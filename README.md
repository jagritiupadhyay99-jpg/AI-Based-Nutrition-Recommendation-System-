# 🍏 NutriGenie - AI Diet Planner (by Team Misfits)

Hi! We are **Team Misfits**, and this is our student project: an AI-powered Nutrition Recommendation System. We built this to help people easily get personalized diet plans and meal recommendations based on their body type, goals, and health conditions. 

🔗 **[Try the Live App Here!](https://diet-planner-using-ai-assistance.vercel.app/)**

## 🚀 What It Does
Our web app takes in basic details (like your height, weight, food preferences, and allergies) and uses AI to generate a fully customized meal plan in seconds. It also features a built-in AI chatbot to answer any extra nutrition questions you might have on the spot!

## 🛠️ Built With
* **HTML, CSS, JavaScript** (Vanilla frontend for a lightweight experience)
* **Groq API** (for super fast AI processing)
* **LLaMA-3 70B Model** (the brains behind the diet recommendations)

## 💡 Why We Built This
As students, we noticed that getting a personalized diet plan usually costs a lot of money or requires hours of research. We wanted to build something free, fast, and easy to use for everyone! 

## ⚙️ How to Run Locally
If you want to download and run this code on your own machine:
1. Clone the repository.
2. Open `scriptofnutrition.js`.
3. Replace the placeholder `"YOUR_GROQ_API_KEY"` with your own free Groq API key.
4. Open `index.html` in your browser.
